# Aardvark — Быстрый старт на Windows

Инструкция позволяет запустить ретрансляцию Telegram-каналов в Delta Chat и/или на электронную почту за несколько минут.

---

## Что потребуется

1. **Пакет Aardvark** — скачанный и распакованный в любую папку
2. **Учётные данные Telegram API** — API ID и API Hash
3. **Адрес электронной почты** — для доставки через Delta Chat или прямой почтовой рассылки (при необходимости)
4. **Имена Telegram-каналов** — которые нужно ретранслировать

---

## Шаг 1 — Получите учётные данные Telegram API

1. Откройте <https://my.telegram.org/apps> в браузере
2. Войдите с номером телефона учётной записи Telegram
3. Создайте приложение (название может быть произвольным, например «Aardvark»)
4. Скопируйте значения **api_id** (числовой идентификатор) и **api_hash** (32-значная строка)

---

## Шаг 2 — Подготовьте почтовый аккаунт *(при необходимости)*

Если планируется доставка через Delta Chat или прямая почтовая рассылка, подготовьте
**отдельный почтовый адрес** только для рассылки (не личную почту).

Перед запуском установщика подготовьте следующее:

| Что | Пример |
|---|---|
| Адрес отправителя | `relay@example.com` |
| Пароль или **пароль приложения** | (см. ниже) |
| **IMAP-сервер** | `imap.yandex.ru` |
| **SMTP-сервер** | `smtp.yandex.ru` |

**Пароль приложения** требуется некоторыми почтовыми сервисами при включённой двухфакторной аутентификации (2FA):
- **Яндекс** — сначала включите IMAP в настройках почты, затем создайте пароль приложения, если 2FA включён
- **Gmail** — пароль приложения необходим при включённой 2FA: Google Аккаунт → Безопасность → Пароли приложений
- **Fastmail, Mailbox.org** — обычный пароль работает

Типовые адреса серверов:

| Провайдер | IMAP | SMTP |
|---|---|---|
| Яндекс | `imap.yandex.ru` | `smtp.yandex.ru` |
| Gmail | `imap.gmail.com` | `smtp.gmail.com` |
| Fastmail | `imap.fastmail.com` | `smtp.fastmail.com` |
| Mailbox.org | `imap.mailbox.org` | `smtp.mailbox.org` |

---

## Шаг 3 — Запустите установщик

Откройте папку с распакованным пакетом и дважды щёлкните:

```
installers\windows\install.cmd
```

Или выполните из Командной строки (от имени администратора):

```cmd
installers\windows\install.cmd
```

Установщик запустит интерактивный мастер настройки.  Введите:

- **API ID** и **API Hash** (из шага 1)
- Номер телефона учётной записи Telegram
- Имена или идентификаторы каналов для ретрансляции — например `@channelname` или числовой ID
- Параметры Delta Chat и/или почтовой рассылки (при необходимости)

Остальные параметры можно оставить по умолчанию.

---

## Шаг 4 — Подтвердите вход в Telegram

При первом запуске Telegram отправит **SMS-код подтверждения** на указанный номер телефона.
Введите его в терминале установщика.

Если в учётной записи включена **двухфакторная аутентификация (Cloud Password / 2FA)**,
сразу после ввода SMS-кода появится запрос пароля — введите его там же.

После успешного входа сессия сохраняется в файл `.session`.
При последующих запусках служба использует сохранённую сессию автоматически — повторный ввод кода не требуется.

---

## Шаг 5 — Поделитесь ссылками-приглашениями Delta Chat

После запуска службы ссылки-приглашения на Delta Chat-каналы появятся в файле:

```
C:\Program Files\Aardvark\invite_links.txt
```

Передайте эти ссылки подписчикам через **защищённый канал связи**
(например, Signal или зашифрованную почту).
Для начала получения сообщений получатели должны открыть ссылку в приложении Delta Chat.

---

## Управление службой

```cmd
sc query   AardvarkRelay
sc start   AardvarkRelay
sc stop    AardvarkRelay
```
